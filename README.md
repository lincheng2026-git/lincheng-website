# 林澄 | Lin Cheng — 个人网站

基于 PRD V2.0 开发的个人审美型网站，技术栈：Next.js 14、Tailwind CSS、Framer Motion。

## 功能概览

- **首页**：全屏 Hero、今日灵感、栏目入口、最近更新、精选作品 Lightbox
- **关于**：杂志式双栏排版
- **灵感日志**：分类筛选（URL 参数）、Markdown 文章详情、上下篇导航
- **猫咪宇宙**：壁纸 / 日记 / 观察 标签切换
- **宋式美学**：配色色卡复制、东方空间与情绪卡片
- **手帐研究所**：杂志式图文、Lightbox 组图
- **器物与瓷器**：大图横幅 + 器物卡片
- **AI 创作实验室**：Before/After 滑块对比、创作时间轴
- **壁纸收藏馆**：分类筛选、Masonry 瀑布流、Lightbox
- **审美档案馆**：按年份归档
- **联系方式**：邮箱一键复制

## 本地开发

```bash
npm install
npm run dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)。

## 部署

推荐部署至 [Vercel](https://vercel.com)，与 Next.js 原生集成。
